# Guide d’onboarding

## 1. Cloner le repo

    git clone <repo_url>

## 2. Installer les dépendances

    pip install -r requirements.txt

## 3. Lancer les tests

    pytest

## 4. Lancer l’API (exemple)

    python src/main.py

## 5. Générer la doc

    cat DOC.md

## 6. CI/CD

    Voir .github/workflows/ci.yaml

## 7. Swagger

    Utiliser openapi.yaml avec Swagger UI

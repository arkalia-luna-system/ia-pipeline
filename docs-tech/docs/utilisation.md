# Utilisation rapide

## Lancer le menu CLI
```bash
python3 -m athalia_core.main
```

## Automatiser (Task)
```bash
task build   # Générer un projet IA
task test    # Lancer tous les tests
task clean   # Nettoyer tous les artefacts
task audit   # Audit sécurité sur tous les projets
```

## Tests
```bash
pytest tests/
```

## Export
```bash
task export
```
